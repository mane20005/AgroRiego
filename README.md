# AgroTech
# Plataforma Inteligente de Optimización de Riego Agrícola
es una plataforma web inteligente que ayuda a pequeños y medianos productores agrícolas de Santa Cruz a optimizar el uso del agua y la energía utilizada 
en sistemas de riego mediante análisis climático, cálculos agronómicos y recomendaciones generadas con inteligencia artificial.
